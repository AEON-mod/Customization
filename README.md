🌌 AEON-mod: PC Aesthetics & UI Ricing
A collection of configuration files, scripts, and tweaks to transform your desktop into a clean, cinematic workspace. This repository focuses on transparency, blur effects, and minimalist UI components.

📁 Repository Structure
| Folder | Description |
|---|---|
| App Transparency Script | Scripts to enable transparency/opacity on various Windows applications. |
| Explorer Blur-Mica | Tweaks to add Mica or Blur effects to the Windows File Explorer. |
| Font Change Commands | PowerShell or CMD scripts to quickly swap system fonts. |
| RainMeter | Custom skins or layouts for the Rainmeter desktop customization tool. |
| Terminal | Configs for Windows Terminal (including config.ps1 for shell styling). |
| Windhwak | Mods and scripts for the Windhawk customization engine. |
| Yasb | Settings for Yet Another Status Bar (YASB). |

🎨 Visual Identity
 * Theme: Moody, high-contrast aesthetics.
 * Recommended Font: JetBrains Mono Nerd Font (or your preferred programming font).
 * Colors: Optimized for dark modes and blurred backgrounds.

🚀 How to Use
 * Clone the repo:
   git clone https://github.com/YourUsername/AEON-mod.git
 * Explore Folders: Each directory contains a specific component. Check the README.md within individual folders (where available) for specific setup instructions.
 * Requirements: Most tweaks require tools like Windhawk, Rainmeter, or YASB already installed on your system.
