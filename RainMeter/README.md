# Raunmeter Skins

Custom skins for Raunmeter – personalize your experience with unique visual themes!

## 📥 Download Instructions

1. **Choose your preferred skin** from the releases or skins folder
2. **Download the skin file** (usually a `.zip` or `.skin` file)
3. **Extract the file** if it's compressed
4. **Locate your Raunmeter installation folder**:
   - Windows: `C:\Program Files\Raunmeter\` or `C:\Users\[YourUsername]\AppData\Local\Raunmeter\`
   - Mac: `/Applications/Raunmeter/` or `~/Library/Application Support/Raunmeter/`
5. **Copy the skin file/folder** into the `skins` directory inside the Raunmeter folder
6. **Launch Raunmeter** and open the settings
7. **Go to the "Skins" or "Appearance" section**
8. **Select your newly installed skin** from the dropdown/list
9. **Click "Apply"** and restart Raunmeter if prompted

## 🛠️ How to Create Your Own Skin

1. Copy an existing skin file as a template
2. Edit the configuration file (usually `skin.json` or `config.xml`)
3. Modify colors, fonts, and layout settings
4. Test your changes in Raunmeter
5. Share your creation with the community!

## ⚠️ Troubleshooting

- **Skin not showing up?** Make sure the file is in the correct `skins` folder
- **Raunmeter crashes after applying skin?** Revert to default skin in safe mode
- **Colors look wrong?** Update to the latest version of Raunmeter
- **Need help?** Open an issue or visit our [Discord server](#)
